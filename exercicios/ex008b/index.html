Código-fonte e pré-formatação

uma forma de definir linhas de código é usando o code

print('Olá, mundo') é um exemplo.

Muitas vezes é interessante que o código tenha cara de código, com as identações características. No caso do python, elas são obrigatórias.
Nesse caso, podemos usar a tag pre, de pré-formatação, que mantém todos os espaços vazios do código.

            
    n = int(input('Digite um número: '))
        if n % 2 == 0:
            print(f'O número {n} é par.')
        elif:
            print(f'IO número {n} é ímpar.')

Citações

Como diria um pai de um amigo do Guanabara: O computador é um burro muito rápido. Essa citação (quotation) é feita com a tag q.

Novamente não devem ser colocadas aspas somente, para manter a semântica. Da mesma forma, caso as aspasdevem ser usadas fora de uma citação (um neologismo, por exemplo), não deve-se usar o a tag q.

Outra opção é fazer citações completas, como cita-se livros e artigos, com a citação em blockquote.

Não acredite em tudo que você lê na internet. - Pedro Álvares Cabral

Ao citar em blockquote e, principalmente, disponiblizar o link no código, os navegadores e mecanismos de pesquisa conseguem referenciar melhor o sei site e mostrá-lo mais nas pesquisas.

Abreviações

É possível criar abreviações, explicando o significado de siglas, que aparecem no navegador se você colocar o mouse em cima. Por exemplo, OQFNB. Esse sublinhado ESCROTO pode-se remover com CSS.
