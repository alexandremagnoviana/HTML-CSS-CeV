Tags semânticas

É importante no HTML5 que haja uma diferença muito clara entre semântica e forma.

Tags semânticas

As tags semânticas são aquelas que tem um significado. as tags strong e emphasis (em), por exemplo, possuem um significado.

Tags de forma

As tags de forma, por sua vez, não trazem nenhum significado. O negrito (b) e o italico (i) trazem somente formas, mas não informação sobre o conteúdo daquilo.

Por que isso importa?

Primeiramente, porque o Google tende a não mostrar muitos resultados de sites com tags antigas. Isso já é motivo suficiente.
Segundo, alguns realmente trazem mais informações para o navegador. Por exemplo, a tag adress pode já levar o usuario ao google maps caso ele clique, o que não aconteceria com um itálico, sublinado ou negrito.

Texto deletado

É possível marcar um texto excluído com a tag del.

Texto inserido

Da mesma forma, pode-se marcar um texto como inserido com a tag ins. Note que o resultado é o mesmo de usar o sublinhado com a tag u, mas um é semântico e o outro é apenas de forma.
