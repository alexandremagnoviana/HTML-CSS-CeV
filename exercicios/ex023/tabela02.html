Tabelas grandes

Para tabelas grandes, é interessante que se use outra configuração semântica/anatômica. Isso é muito importante para mecanismos de busca.

Iserimos a cabeça da tabela com o thead;
depois o corpo do com o tbody
e finalmente o pé com o tfoot

A tag th seria para títulos nas tabelas, equivalente a h1 a h6 no corpo do site. A tag td é para dados. Deve semper haver pelo menos um td relativo a um th. Para identificar onde estam os dados relativos àquele th estão usa-se scope="col" ou "row"

A tag caption é usada para criar o título da tabela
