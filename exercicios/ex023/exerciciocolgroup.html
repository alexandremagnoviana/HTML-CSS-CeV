Destaque para colunas

Para dar destaque a uma linha é muito simples, já que toda a linha obrigatoriamente pertence a uma mesma tag tr e, portanto, basta criar uma classe nessa tag e fi de faina.

Porém, dar destaque a uma coluna não é tão simples, especialmente para tabelas com muitas linhas, pois dever-se-ia atribuir uma classe a cada elemento, o que é impraticável em tabelas muito grandes e de difícil manutenção.

Para resolver isso pode-se usar o artifício do colgroup, dentro da tag table. Dentro da tag colgroup é possível criar uma tag col para cada coluna e então atribuir classes a ela.
